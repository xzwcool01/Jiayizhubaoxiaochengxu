# 项目启动指南

---

## 1. jiayi-api（IntelliJ IDEA）

### 首次打开
1. 打开 IDEA → `File` → `Open...`
2. 选择 `E:\AiProject\Jiayizhubaoxiaochengxu\jiayi-api`
3. 提示"Maven projects found" → 点 **Load**（等待右下角索引完成）

### 配置 JDK 17
1. `File` → `Project Structure` (`Ctrl+Shift+Alt+S`)
2. `Project` → `SDK` → 点 `Add SDK` → `JDK`
3. 浏览到 `C:\Program Files\Java\jdk-17.0.1` → 确定
4. `Project language level` 选 `17`
5. 点 OK

### 配置 Maven
1. `File` → `Settings` → `Build, Execution, Deployment` → `Build Tools` → `Maven`
2. `Maven home path` → 选 `D:\ruanjian\apache-maven-3.9.9`
3. `User settings file` → 勾选 Override → 选 `D:\ruanjian\apache-maven-3.9.9\conf\settings.xml`
4. `Local repository` → 勾选 Override → 设一个路径（如 `D:\.m2\repository`）
5. 点 OK

### 启动
- **方式 A**：在右侧 Maven 面板 → `Lifecycle` → 双击 `spring-boot:run`
- **方式 B**：打开 `JiayiApplication.java` → 点行号旁的绿色三角 → `Run`
- 启动后访问：http://localhost:8080/api/health
- 预期返回：`{"code":200,"message":"success","data":"Hello 嘉怡珠宝"}`

### 停止
点 IDEA 左上角红色方形按钮，或 `Ctrl+F2`

## 2. jiayi-admin（VS Code）

### 首次打开
1. 打开 VS Code
2. `File` → `Open Folder...` → 选 `E:\AiProject\Jiayizhubaoxiaochengxu\jiayi-admin`
3. 等待右下角 npm 依赖加载完成

### 启动
1. `Terminal` → `New Terminal`（或 `` Ctrl+` ``）
2. 输入：
```bash
npm run dev
```
3. 看到输出：
```
VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://localhost:5173/
```
4. `Ctrl+点击` http://localhost:5173 打开浏览器

### 登录
- 浏览器打开 http://localhost:5173
- 自动跳转到登录页
- 输入任意账号密码 → 点"登录"（当前是模拟登录）
- 进入后台布局页面（左侧菜单 + 6个模块空路由）

### 停止
在 VS Code 终端按 `Ctrl+C`，输入 `y` 回车

### 推荐安装的 VS Code 插件
| 插件 | 用途 |
|------|------|
| Vue - Official | Vue3 语法高亮/提示 |
| ESLint | 代码检查 |
| TypeScript Vue Plugin | TS 增强 |

## 3. jiayi-miniapp（HBuilderX）

### 首次打开
1. 打开 HBuilderX（`D:\ruanjian\HBuilderX\HBuilderX.exe`）
2. `文件` → `打开目录...` → 选 `E:\AiProject\Jiayizhubaoxiaochengxu\jiayi-miniapp`
3. 第一次会提示安装依赖 → 点 **是**（自动 npm install）

### 检查 AppID
1. 在左侧文件树找到 `src/manifest.json`
2. 点开 → 找到 `mp-weixin` → `appid` 项
3. 确认是：`wxff03445f8c883241`
4. 如果不是，改成你自己的 AppID

### 启动
**前提：必须先启动 jiayi-api（后端 API）**

#### 方式一：微信开发者工具（推荐）
1. 顶部菜单 → `运行` → `运行到小程序模拟器` → `微信开发者工具`
2. 首次需要配置微信开发者工具路径：
   - 如果弹出提示 → 点"设置" → 找到 `微信web开发者工具.exe` 路径
   - 如果之前没装，下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
3. 等待编译完成，微信开发者工具会自动打开

#### 方式二：H5 浏览器预览（不需要微信开发者工具）
1. 顶部菜单 → `运行` → `运行到浏览器` → 选 Chrome
2. 浏览器打开后可以看到 Tab 页面（但部分微信API不可用）

### 验证
- 底部 5 个 Tab：首页 / 分类 / 发现 / 会员 / 我的
- 点击每个 Tab 切换到对应页面
- 如果开通了后端 API，首页会显示从后端获取的数据

### 停止
- 微信开发者工具：直接关窗口
- HBuilderX：顶部红色方形按钮停止运行

## 常见问题

| 问题 | 解决 |
|------|------|
| IDEA 打开后 pom.xml 爆红 | 等右下角 Maven 下载完成，或点右侧 Maven → 刷新按钮 |
| VS Code npm run dev 报错 | 检查是否在 jiayi-admin 目录下，重新 `npm install` |
| HBuilderX 提示"没有安装微信开发者工具" | 设置路径到微信开发者工具的安装位置 |
| 后端接口 404 | 确认 jiayi-api 先启动成功 |
| 后端报数据库连接失败 | 检查 MySQL 服务是否在运行，密码是否正确 |
