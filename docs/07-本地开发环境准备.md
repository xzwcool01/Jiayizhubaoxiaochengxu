## 三、JDK 安装要点

```bash
# 安装完成后设置环境变量：
# 此电脑 → 属性 → 高级系统设置 → 环境变量

# 新建 JAVA_HOME
# 变量名: JAVA_HOME
# 变量值: C:\Program Files\Java\jdk-17（你的安装路径）

# 编辑 Path 变量，添加:
%JAVA_HOME%\bin

# 验证安装
java -version
# 应显示: openjdk version "17.x.x"
```

## 四、Maven 安装要点

```bash
# 1. 下载 apache-maven-3.9.x-bin.zip
# 2. 解压到 D:\software\apache-maven-3.9.x
# 3. 设置环境变量 MAVEN_HOME = D:\software\apache-maven-3.9.x
# 4. Path 添加 %MAVEN_HOME%\bin

# 验证
mvn -v
# 应显示 Apache Maven 3.9.x
```
