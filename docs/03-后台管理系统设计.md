## 1. 技术架构

### 1.1 整体架构

```
┌─────────────────────────────────────────────────┐
│                  前端层 (Vue 3)                   │
│  ┌──────────┐  ┌──────────┐  ┌──────────────┐   │
│  │ PC Web端  │  │ 移动H5端  │  │  ECharts图表  │   │
│  │ Element+  │  │ 自适应    │  │  统计可视化   │   │
│  └──────────┘  └──────────┘  └──────────────┘   │
├─────────────────────────────────────────────────┤
│                 API层 (RESTful)                   │
│  ┌──────────┐  ┌──────────┐  ┌──────────────┐   │
│  │ Spring   │  │ Spring   │  │  微信支付    │   │
│  │ Boot API │  │ Security │  │  第三方API   │   │
│  └──────────┘  └──────────┘  └──────────────┘   │
├─────────────────────────────────────────────────┤
│                 数据层                            │
│  ┌──────────┐  ┌──────────┐  ┌──────────────┐   │
│  │ MySQL    │  │ Redis    │  │  对象存储COS  │   │
│  │ 主数据库  │  │ 缓存(后期)│  │  图片/文件    │   │
│  └──────────┘  └──────────┘  └──────────────┘   │
└─────────────────────────────────────────────────┘
```

### 1.2 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Vue 3 + Vite | 高性能前端框架 |
| UI组件库 | Element Plus | 企业级后台组件库 |
| 图表 | ECharts 5 | 数据可视化 |
| 后端框架 | Spring Boot 3.x | Java微服务框架 |
| ORM | MyBatis-Plus | 数据库操作 |
| 权限 | Spring Security + JWT | 认证授权 |
| 数据库 | MySQL 8.0 | 关系型数据库 |
| 缓存 | Redis (后期按需) | 热点数据缓存 |
| 文件存储 | 腾讯云COS | 图片/文件存储 |

## 4. API接口设计

### 4.1 通用规范

- **基础路径**：`/api/v1`
- **请求格式**：JSON
- **响应格式**：
```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": 1234567890
}
```
- **认证方式**：`Authorization: Bearer {token}`
- **分页参数**：`?page=1&size=20`
- **分页响应**：
```json
{
  "records": [],
  "total": 100,
  "size": 20,
  "current": 1,
  "pages": 5
}
```
