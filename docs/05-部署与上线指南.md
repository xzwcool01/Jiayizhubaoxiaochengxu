## 1. 服务器选购

### 1.1 推荐配置方案

#### 初期方案（推荐，性价比最高）

| 项目 | 配置 | 预估费用 |
|------|------|---------|
| 腾讯云轻量应用服务器 | 2核2G / 5M带宽 / 80G SSD | ~450元/年（新用户首年） |
| MySQL数据库 | 同机部署（暂不单独购买） | 包含在服务器中 |
| 腾讯云COS | 对象存储（图片等静态资源） | 按量付费，月均约10元 |
| 域名 | .com/.cn | ~50元/年 |

**首年总成本约：510元**

#### 进阶方案（日活5000+）

| 项目 | 配置 | 预估费用 |
|------|------|---------|
| 腾讯云CVM | 4核8G / 5M带宽 | ~1000元/年 |
| 云数据库MySQL | 2核4G | ~1200元/年 |
| 腾讯云COS | 对象存储 | 按量付费 |
| Redis | 1G基础版（可选） | ~600元/年 |
| 域名 | .com/.cn | ~50元/年 |

### 1.2 选购步骤

1. 访问 [腾讯云官网](https://cloud.tencent.com)
2. 注册账号并完成实名认证
3. 搜索"轻量应用服务器"
4. 选择配置：2核2G、5M带宽、80G SSD
5. 镜像选择：Ubuntu 22.04 LTS 或 CentOS 7.9
6. 购买时长：建议1年（新用户优惠最大）
7. **新用户建议**：先购买1个月测试，确认没问题后再续费

## 2. 域名与备案

### 2.1 域名购买
- 在腾讯云"域名注册"中购买
- 推荐后缀：.com / .cn / .shop
- 命名建议：`jiayi.com` 或 `jiayijewelry.com`（需查询是否可注册）
- 价格：约30-60元/年

### 2.2 ICP备案
**前提**：国内服务器必须完成备案

#### 备案流程
```
1. 腾讯云控制台 → 备案服务
2. 填写主体信息（营业执照信息）
3. 填写网站信息
4. 上传资料（营业执照/法人身份证）
5. 腾讯云初审（1-2个工作日）
6. 短信核验
7. 管局审核（7-20个工作日）
8. 备案完成，获取备案号
```

**注意事项**：
- 备案期间网站不能访问
- 备案需在购买服务器后进行
- 如无备案需求，可选择香港服务器（但延迟稍高）

### 2.3 SSL证书

微信小程序强制要求HTTPS，腾讯云提供免费SSL证书：

```
1. 腾讯云控制台 → SSL证书 → 免费证书
2. 选择域名型免费证书（DV）
3. 提交申请
4. DNS验证（在域名解析中添加TXT记录）
5. 签发成功（通常10分钟内）
6. 下载证书（Nginx格式）
```

## 4. 后端部署

### 4.1 服务器环境搭建

```bash
# 1. 连接服务器（使用SSH工具如Xshell或直接在终端）
ssh root@你的服务器IP

# 2. 更新系统
apt update && apt upgrade -y

# 3. 安装JDK 17
apt install openjdk-17-jdk -y
java -version  # 验证安装

# 4. 安装MySQL 8.0
apt install mysql-server-8.0 -y
mysql_secure_installation  # 安全配置

# 5. 安装Nginx（反向代理）
apt install nginx -y
systemctl start nginx
systemctl enable nginx

# 6. 安装Git（拉取代码）
apt install git -y
```

### 4.2 数据库初始化

```bash
# 1. 登录MySQL
mysql -u root -p

# 2. 创建数据库
CREATE DATABASE jiayi_jewelry DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 3. 创建用户
CREATE USER 'jiayi'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON jiayi_jewelry.* TO 'jiayi'@'localhost';
FLUSH PRIVILEGES;

# 4. 导入数据库表结构
mysql -u jiayi -p jiayi_jewelry < /path/to/schema.sql
```

### 4.3 项目配置


### 4.4 打包与部署

```bash
# 1. 使用Maven打包
mvn clean package -DskipTests

# 2. 上传jar包到服务器
# 使用SFTP/SCP上传 target/jiayi-api.jar 到 /opt/jiayi/

# 3. 启动服务
cd /opt/jiayi
nohup java -jar jiayi-api.jar --spring.profiles.active=prod > app.log 2>&1 &

# 4. 验证启动
tail -f app.log  # 查看日志
```

### 4.5 Nginx反向代理配置

```nginx
# /etc/nginx/sites-available/jiayi
server {
    listen 443 ssl;
    server_name api.jiayi.com;  # 替换为你的域名

    ssl_certificate /etc/nginx/ssl/jiayi.crt;
    ssl_certificate_key /etc/nginx/ssl/jiayi.key;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # 文件上传大小限制
    client_max_body_size 20M;
}

# HTTP → HTTPS 重定向
server {
    listen 80;
    server_name api.jiayi.com;
    return 301 https://$server_name$request_uri;
}
```

```bash
# 启用配置
ln -s /etc/nginx/sites-available/jiayi /etc/nginx/sites-enabled/
nginx -t  # 验证配置
systemctl reload nginx
```

## 6. 微信支付接入

### 6.1 申请微信支付

```
1. 微信公众平台 → 功能 → 微信支付
2. 点击"开通"
3. 提交资质审核（营业执照/法人身份证/银行账户）
4. 审核通过后获取商户号（mch_id）
5. 设置API密钥（APIv2密钥/APIv3密钥）
6. 配置支付回调域名
```

### 6.2 开发接入

```java
// 微信支付统一下单
@PostMapping("/pay/unified-order")
public Result unifiedOrder(@RequestBody PayRequest request) {
    // 1. 生成订单号
    String orderNo = generateOrderNo();
    
    // 2. 调用微信支付统一下单API
    Map<String, String> params = new HashMap<>();
    params.put("appid", wxAppId);
    params.put("mch_id", wxMchId);
    params.put("nonce_str", generateNonceStr());
    params.put("body", "嘉怡珠宝-商品购买");
    params.put("out_trade_no", orderNo);
    params.put("total_fee", String.valueOf(request.getPayAmount() * 100)); // 单位:分
    params.put("spbill_create_ip", getClientIp());
    params.put("notify_url", "https://api.jiayi.com/api/v1/pay/notify");
    params.put("trade_type", "JSAPI");
    params.put("openid", request.getOpenId());
    
    // 3. 签名
    String sign = WxPayUtil.generateSignature(params, wxMchKey);
    params.put("sign", sign);
    
    // 4. 调用统一下单
    String result = WxPayUtil.unifiedOrder(params);
    
    // 5. 返回小程序调起支付所需的参数
    return Result.success(result);
}
```

## 8. 常见问题

### Q1: 服务器2核2G够用吗？
初期够用。如果后续用户量增大，可在腾讯云控制台直接升级配置。

### Q2: 备案期间可以开发吗？
可以。备案期间可以使用IP地址 + 小程序开发者工具中的"不校验合法域名"选项进行开发和测试。

### Q3: 微信支付审核多久？
通常1-3个工作日。需要确保提交的资质清晰完整。

### Q4: 小程序审核被拒怎么办？
查看拒审原因（常见问题：类目不符/功能不完整/隐私政策缺失），修改后重新提交。

### Q5: 没有https可以吗？
不行。微信小程序强制要求所有网络请求使用HTTPS。

### Q6: 数据如何备份？
推荐方案：
1. MySQL定时备份：每天凌晨自动备份
2. 备份文件上传到腾讯云COS
3. 保留最近7天备份

```bash
# 数据库自动备份脚本
#!/bin/bash
mysqldump -u jiayi -p'password' jiayi_jewelry > /backup/jiayi_$(date +%Y%m%d).sql
# 上传到COS
coscli cp /backup/jiayi_$(date +%Y%m%d).sql cos://bucket/backup/
# 删除7天前的备份
find /backup -name "*.sql" -mtime +7 -delete
```
