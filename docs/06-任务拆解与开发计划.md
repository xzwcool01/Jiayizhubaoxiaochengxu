# 嘉怡珠宝商城 - 任务拆解与开发计划

## 开发模式说明

- **前后端并行开发**，每个阶段都有可测试的交付物
- 每个阶段约 1-2 周，共 **6 个阶段**
- 每个阶段结束后可独立测试

---

## Phase 0：环境搭建（3天）

| # | 前端任务 | 后端任务 |
|---|---------|---------|
| 0.1 | 创建 uni-app 项目 | 创建 Spring Boot 项目 |
| 0.2 | 配置微信小程序 AppID | 配置 MySQL 数据库 |
| 0.3 | 配置 UI 主题色/全局样式 | 配置 application.yml |
| 0.4 | 配置底部 Tab 导航（5个空壳页面） | 配置 MyBatis-Plus + 自动建表 |
| 0.5 | - | 搭建腾讯云服务器环境 |

**可测试**：前端显示5个Tab页面可切换；后端启动成功无报错
